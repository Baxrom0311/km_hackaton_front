# KM PostureAI Landing

Bu repo `KM PostureAI` uchun tozalangan landing page.

Asosiy o'zgarishlar:
- generated sectionlar va Lovable tagger olib tashlandi
- sahifa bitta aniq product-story asosida qayta yig'ildi
- styling yengillashtirildi va professional landing ko'rinishiga o'tkazildi

Ishga tushirish:

```bash
npm install
npm run dev
```

Tekshiruv:

```bash
npm run build
npm run lint
npm test
```
